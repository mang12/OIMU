Oneday I Met You

오이뮤는 2015년부터 서울을 기반으로 활동하고 있는
라이프스타일 브랜드이자, 디자인 스튜디오입니다.
우리는 시대정신을 갖고 과거와 현재의 가치를 잇는 디자인 활동을 하며,
다양한 인쇄매체와 제품, 공간, 전시, 콘텐츠 퍼블리싱 등을
통해 디자인을 제안합니다.

OIMU is a Seoul based design studio established in 2015.
We believe that design takes on the role of a bridge that connects
the value of the past with present.
The OIMU projects reinterpret the forgotten cultural value.
We create brand identities, printed matter, products, artwork,
exhibition, content publishing and more.

Project 1. Match project

it is nostalgic to the older generation and presents unfamiliar
charm of the matches to the younger generation.

Project 2. Air project

We announce the traditional culture of the incense
and create beautiful scenery where good fragrance is shared.

Project 2.5. Nobang project

We propose a new use of Nobang in various forms,
breaking away from the limited use of traditional clothing.

Project 3. Fortune project

The objective of Fortune project is to reinterpret the objects
that symbolized fortune in a modern way in the ethnic culture
of desiring for fortune.

Project 4. Eraser project

We organized a list of 453 erasers which was produced
by Hwarang since 1950 in Korea.
We hope this would act as a bridge to
find the value of small eraser.

Project 5. Color project

We compiled a book that defines the color names
and specific colors of natural objects
we meet every day.
